App Creation Prompt

Build a colorful chore and task tracking app for 2 kids as a single downloadable HTML file that works as a web app on android phone/ipad (saved to home screen via any browser). Name it "Dallas Palace Chore Tracker". Next to the Settings button, add a tab named "Prompt" where the below prompt is pasted and nicely formatted - for anyone who wants to recreate the app using this prompt. Layout & Design • Two separate kid dashboards as tabs at the top • Tab labels should show the kid's actual name (customizable) • Theme with bright, colorful kid-friendly design • Each chore displayed as a colorful card with a toggle switch • Cumulative points displayed at the top of each kid's dashboard Chores & Points • Default starter chores: Make bed (5pts), Help with dishes (5pts), Practice (10pts), Exercise/Play outdoors (10pts), Tidy room (10pts), Cleanup after eating (5pts), Help cleaning living room (10pts), Homework (15pts) • Tapping a chore card toggles it on/off and adds/deducts points accordingly • A manual point adjustment field on each kid's page — type any positive or negative number and tap Apply to instantly update their total Penalties • A separate Penalties section in Settings where you can add negative-point items if needed (e.g. not eating vegetables/fruits -10 pts, too much screen time -20 pts) Settings Panel • Rename each kid (names update everywhere including tabs) • Add, edit, and delete chores with custom names and point values • Add, edit, and delete penalty items with custom names and point values • Total Reset should happen only with confirmation • A separate "Reset today's switches" button on each kid's dashboard that clears daily toggles without wiping cumulative points Technical Requirements • Single self-contained HTML file, no external dependencies • Uses localStorage to persist all data (points, names, chores, today's switches) between sessions • Includes apple-mobile-web-app-capable meta tags so it works cleanly when added to iPad home screen
